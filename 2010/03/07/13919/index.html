[.NET Resource].NET Framework Setup Verification Tool

Posted on 2010-03-07   |  

.NET Framework Setup Verification Tool是一套驗證軟體，會驗證.NET Framework的安裝檔案、目錄、登錄檔，也會使用簡單的小程式來測試.NET Framework是否可以被運行。

目前支援的.NET Framework版本有：

  • .NET Framework 1.0
  • .NET Framework 1.1
  • .NET Framework 1.1 SP1
  • .NET Framework 2.0
  • .NET Framework 2.0 SP1
  • .NET Framework 2.0 SP2
  • .NET Framework 3.0
  • .NET Framework 3.0 SP1
  • .NET Framework 3.0 SP2
  • .NET Framework 3.5
  • .NET Framework 3.5 SP1

使用上只要先選取.NET Framework的版本。

按下[Verify Now]按鈕。

程式會開始做驗證的動作。

驗證完後，可在Current status看到驗證的結果。

另外一提，經過測試，這套工具的防呆部分感覺有點沒做好，產品版本可以手動輸入，輸入後按下[Verify Now]會有不正確的結果。

更多詳細的內容，請自行參閱該軟件壓縮檔內的ReadMe.Txt。
