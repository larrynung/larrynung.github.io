[.NET Resource]Visual Studio Color Theme Editor

Posted on 2010-06-24   |  

Visual Studio Color Theme Editor是Visual Studio的配色外掛，能編輯外觀設定檔、匯入/匯出設定檔、套用外觀設定檔。可在Visual Studio Color Theme Editor網站下載點擊安裝，或是透過Visual Studio 2010內建的Extension Manager來安裝。

安裝完畢後重開，可在上方主選單上發現多了一個Theme的選單。點選下拉可發現預設有八個預設外觀可供我們套用。

外觀畫面如下：

Windows Aero Windows Classic
Windows XP Blue Windows XP Olive
Windows XP Silver Autumn
Emerald Silver

若預設的外觀不足以滿足個人的需求，可透過滑鼠點選[Theme]→[Customize Colors…]開啟Customize Colors對話盒，對話盒左上方工具列按鈕依序可新增、開啟、儲存、刪除外觀設定檔。右上方的工具列按鈕則是可以複製、貼上、復原、取消復原、調整色調/飽和度/亮度、與調整色彩。

在新增外觀設定檔時，會需要指定外觀設定檔的名稱，以及其所預複製的外觀設定來源，確定後新的外觀設定會從指定的外觀設定檔複製，預設的設定值都會跟所指定的外觀設定檔一樣。

接著可一項項的針對想變更的顏色做調整。

或是透過按住Ctrl按鈕或是Shift按鈕，選取多個顏色設定。

再按下上方工具列的設定按鈕，批次調整外觀顏色。
