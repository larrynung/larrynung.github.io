要使用 Photo Editor 在 Android 上進行圖片的裁切。  

<!-- More -->

<br/>


可先選取要處理的圖片，選取分享。  

![1.jpg](1.jpg)

<br/>


將圖片分享至 Photo Editor。  

![2.jpg](2.jpg)

<br/>


然後在 Photo Editor 下方找到裁切按鈕。  

![3.jpg](3.jpg)

<br/>


按下裁切按鈕後框出裁切後要留下來的部份，按下右上方的勾勾按鈕確認裁切。  

![4.jpg](4.jpg)

<br/>


圖片會被裁切成只剩下剛所框出的部份。  

![5.jpg](5.jpg)

<br/>


最後按下左上方的箭頭，儲存裁切的結果即可。  

![6.jpg](6.jpg)
