要透過 SQLiteStudio 建立 SQLite 資料庫，可點選 [Database | Add a database] 主選單選項，或是按下熱鍵 Ctrl + O。  

<!-- More -->

![1.png](1.png)

<br/>


在開啟的對話框中，點選 File 後面的 Add 按鈕。  

![2.png](2.png)

<br/>


設定要建立的 SQLite 資料庫存放的位置與檔名，按下 Save 按鈕。  

![3.png](3.png)

<br/>


![4.png](4.png)

<br/>


指定的 SQLite 資料庫檔即會被建立。  

![5.png](5.png)
