Cake 0.30.0 後開始支援 Global .NET CLI tool，可透過 dotnet tool 安裝。  

<!-- More -->

    dotnet tool install -g Cake.Tool

![1.png](1.png)

<br/>



安裝完就可以直接透過 dotnet 命令運行 Cake。  

    dotnet cake <CakeFile>

![2.png](2.png)
