使用 Azure portal 建立函數應用程式，要先選取建立資源。  

<!-- More -->

![1.jpg](1.jpg)

<br/>


在計算資源這邊找到函數應用程式。  

![2.jpg](2.jpg)

<br/>


輸入資源的名稱、資源群組、作業系統、主機位置、儲存體等資訊後按下建立按鈕。  

![3.jpg](3.jpg)

<br/>


建立完成後，所建立的函數應用程式可在資源群組中查閱。
