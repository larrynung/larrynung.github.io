如要在 MAC 上使用 Percona Toolkit，可透過 HomeBrew 進行安裝。  

<!-- More -->

    brew install percona-toolkit

![1.png](1.png)

<br>


安裝後可調用 Percona Toolkit 命令查驗，，確認安裝無誤。  

    pt-variable-advisor --version
