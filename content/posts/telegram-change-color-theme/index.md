---
title: "Telegram - Change color theme"
date: "2019-03-19 07:17:57"
---


要調整 Telegram 的配色，可點選 Telegram 左上腳的按鈕展開側邊選單。  

<!-- More -->

![1.jpg](1.jpg)

<br/>


點選 Settings 進入設定畫面。  

![2.jpg](2.jpg)

<br/>


點選 Chat Settings 進入聊天設定畫面。  

![3.jpg](3.jpg)

<br/>


在 Color theme 那邊選取要使用的 Color theme。  

![4.jpg](4.jpg)

<br/>


即可調整 Telegram 的配色。  

![5.jpg](5.jpg)
