要透過 FastoRedis 查找 Redis 的 Key，可在 Redis 連線後在左側的樹狀節點上面按下滑鼠右鍵，選取滑鼠右鍵快顯選單中的 View keys 選單選項。

<!-- More -->

![1.png](1.png)

<br/>


在彈出的視窗中輸入要用來過濾 Key 的 Pattern，按下 Search 按鈕。

![2.png](2.png)

<br/>


就可以查找 Redis 內符合過濾 Pattern 的 Key。
