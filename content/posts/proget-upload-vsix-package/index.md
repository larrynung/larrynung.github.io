要從本地上傳 VSIX 套件到 ProGet 的 VSIX feed，可在 ProGet 的 VSIX feed 頁面按下 Add Package 按鈕。  

<!-- More -->

![1.png](1.png)

<br/>


點選 Upload Package。  

![2.png](2.png)

<br/>


按下 Choose File 按鈕。  

![3.png](3.png)

<br/>


選取要上傳的 VSIX 套件。

![4.png](4.png)

<br/>


按下 Upload 按鈕。  

![5.png](5.png)

<br/>


選取的套件即會上傳到 ProGet 的 VSIX feed。  

![6.png](6.png)
