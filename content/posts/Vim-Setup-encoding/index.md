使用 Vim 時若因編碼導致文字無法正常顯示。  

<!-- more -->

![1.jpg](1.jpg)

</br>


可開啟編輯 Vim 的設定檔。  

    vi ~/.vimrc

![2.jpg](2.jpg)

</br>


加上 encoding 設定後存檔。  

    set encoding = <Encoding>

![3.jpg](3.jpg)

</br>


Vim 就可以用正確的編碼顯示文字。  

![4.jpg](4.jpg)
