要使用 pip 安裝 Jupyter，可調用下列命令。  

<!-- More -->

    python -m pip install --upgrade pip
    python -m pip install jupyter

<br/>


![1.png](1.png)
 
<br/>


安裝完調用下列命令運行 Jupyter。  

    jupyter notebook


![2.png](2.png)
 
<br/>
