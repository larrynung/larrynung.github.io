如要在 MAC 上使用 Vault，可透過 HomeBrew 進行 Vault 的安裝。  

<!-- More -->

    brew install vault

![1.png](1.png)

</br>


安裝後調用 Vault 命令並帶入 --version 參數，若安裝正常應可查驗到安裝的 Vault 版本。  

    vault --version
