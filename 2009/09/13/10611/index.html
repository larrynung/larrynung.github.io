[WLW]Polaroid Picture

Posted on 2009-09-14   |  

Introduction

Polaroid Picture是Windows Live Writer的外掛，其可以為您插入圖片，並添加特效，像是旋轉、文繞圖、圖片標題等。

安裝步驟

到微軟Live Gallery的Polaroid Picture頁面下載檔案後安裝。

安裝完成後，開啟Windows Live Writer，即會看到外掛已加入Windows Live Writer。

使用步驟

Step1.點選右下角的外掛選項

Step2.在彈出的對話框中選曲要使用的圖片

選取後會出現圖片會插入，並套用像相片一樣的效果。

Step3.再右側設定區設定細部參數

設定完後圖片效果會依設定而變換。

Sysrem Requirement

  • .NET Framework v2.0
  • Windows Live Writer
