[WLW]WebSnapr Plugin

Posted on 2009-09-13   |  

Introduction

WebSnapr是Windows Live Writer的外掛，其功能為插入網站縮圖與連結。

安裝步驟

下載下來後，把檔案解壓縮放到Windows Live Writer安裝路徑下的Plugs目錄即可。

檔案放好後開啟Windows Live Writer，即可在右下角發現該外掛已經加入。

使用步驟

Step1.點選右下角的外掛選項

Step2.在彈出的設定視窗設定網址位置、標題、大小等設定

按下成果Ok後即會看到縮圖

Sysrem Requirement

  • .NET Framework v2.0
  • Windows Live Writer
