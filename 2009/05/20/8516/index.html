[Investment]房仲好、壞識人術

Posted on 2009-05-21   |  

Introduction

以下內容轉載自2009/5/16自由時報週末生活版。

測試房仲專業度

第一招：對房屋市場的敏銳度

好房仲 => 會客觀且條理分析經濟環境與市場趨勢。

壞房仲 => 只會不斷強調自己手上房子最多、最便宜，企圖遮掩缺乏專業的事實。

第二招：是否深入了解物件

好房仲 => 包括格局、面積、房屋瑕疵、裝潢等各方面屋況，通常都清楚掌握並告知。

壞房仲 => 必須不斷查詢資料，或催促買方親自看屋。

第三招：對區域與周遭環境熟悉度

好房仲 => 是否有公園綠地要開發、未來有無重大交通建設等問題，好房仲會查詢資料並詳細告知。

壞房仲 => 一問三不知，事後也不會主動再補充或提醒。

第四招：對房貸成數是否中肯

好房仲 => 會依房屋條件、地點給出適當的房貸成數，依目前景氣約6~8成。

壞房仲 => 信口保證成數要多少有多少，或建議買方去辦高利率的信貸與房屋二胎。

第五招：對稅法了解程度

好房仲 => 房屋買賣牽涉土地稅與契稅等問題，房仲也多少要懂法規，並能提供房屋節稅的方法。

壞房仲 => 會強調不是分內的事，把問題推給買方或代書。

有誠信有保障

第六招：有無告知議價方式

好房仲 => 會明確告知 議價的方式，並說明斡旋金與要約書的不同。

壞房仲 => 避重就輕，催促先下斡旋金好跟屋主殺價。

第七招：有無客觀分析行情

好房仲 => 會依房屋優缺點，提供合理的成交行情。

壞房仲 => 只會強調房屋本身的優勢，開出較高的價格給買方殺價。

第八招：有無主動提醒各種負擔

好房仲 => 如房屋的管理費、契約、代書與財產交易稅等支出，會主動提醒並將疑慮註明在合約中。

壞房仲 => 只會口頭保證，但很可能簽完約就置之不理。

第九招：有無主動說明房屋缺失

好房仲 => 不會隱瞞房子周邊是否有令人嫌惡設施，如電塔、鴿舍等。

壞房仲 => 通常避而不提，真的避不掉時，就改口強調房屋價格超便宜。

好態度更加分

第十招：是否積極幫忙找物件

好房仲 => 會積極了解買方需求後，與手上物件進行配對，或主動開發買方理想屋子。

壞房仲 => 會帶你看難賣的物件，並以低廉價格進行遊說。

第十一招：會不會催促出價

好房仲 => 會給你適當的時間思考，通常會有1~3天。

壞房仲 => 會在現場不斷以言語刺激買方先出價為贏。

第十二招：有沒有大打團體站

好房仲 => 一般房仲是自己帶屋並解說。

壞房仲 => 可能安排其它同事在同一時間帶多組客人去看，自導自演營造搶手氣氛。

第十三招：觀察沒有成交時的態度

好房仲 => 即使這次沒有成功，仍會態度良好，不定時關心或給予最新消息。

壞房仲 => 可能編織各種理由或小道消息，企圖打擾你與其他仲介簽約。

小問題破解房仲術語

第十四招：問「房子多大」

好房仲 => 會具體說：「扣掉停車位、陽台與其它設施後是N坪」的實際回覆。

壞房仲 => 以「房子格局很好，可使用的空間很大」模糊焦點，可能有虛灌坪數的嫌疑。

第十五招：問「有沒有維修過」

好房仲 => 會誠實表示「房東曾換過xx等東西，但xx需買方再住意」。

壞房仲 => 只以「屋子全都整修好了，不用擔心」逃避潛在問題，如噪音、漏水等。

第十六招：問「附近行情如何」

好房仲 => 會就同條件的物件據實回答。

壞房仲 => 常以「這邊都來賣的很好，很久沒有新物件釋出，先搶先贏」等話術，藉稀有性來引誘你出高價、快速成交。

第十七招：問「原屋主為何要賣屋」

好房仲 => 會老實講出具體理由。

壞房仲 => 常會以屋主不在台灣含糊帶過，或「屋主要移民、趕著賣」等話術引用。

第十八招：問「這屋子賣多久了」

好房仲 => 會誠實回答大概賣了幾天，並告知約有幾組客戶看過。

壞房仲 => 僅強調「才剛釋出不久，但有好幾組客人已預約要看屋，喜歡就要快，好房子不等人」等類似話語，不斷鼓舞誘惑你盡快出價。
