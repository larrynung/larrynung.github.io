[.NET Resource][Visual Studio]The Code Project Add-In for Visual Studio 2008

Posted on 2009-05-28   |  

Intordouction

最近上Code Project突然發現首頁左上方有個VS2008的Add-In，索性抓下裝來玩玩看。使用起來意外的順手，在此隨手記錄一下。

取得方式

可透過Code Project首頁左上方連結進入下載頁面

進入下載頁面後點選連結下載並安裝即可。或是透過本文下方的相關連結直接跳到下載頁面抓取。

使用畫面

下載裝完後開啟VS2008，即可在右半邊找到The Code Project Add-In頁籤。

展開後會看到Code Project Add-In的搜尋介面。

鍵入欲搜尋的關鍵字。

按下搜尋，即可看到搜尋的結果。

點選搜尋的結果，即會開啟對應的頁面。

或是也可以選取字串後按下滑鼠右鍵，點選Search Articles，Plug-In會自動帶入選取字串做搜尋動作。

除此之外，上方的工具選單內也多了個The Code Project的選項群組。

透過工具選單內的Add-In Setting選項，可對Add-In做些細部設定的修改。

Conclusion

整體來說這算是一個不錯的程式輔助工具，但是若能再增加搜尋Google Code、MSDN Code Gallery、與CodePlex的話就太完美了。
