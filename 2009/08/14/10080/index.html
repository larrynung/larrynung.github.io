[Software]XMind 心智圖編輯軟體

Posted on 2009-08-15   |  

Introduction

XMind是用來編輯心智圖的軟體，具有免費版與專業版兩種版本。其操作簡易、圖型漂亮、跨平台。

還支援多種圖型架構像是心智圖

二維表格

魚骨圖

邏輯圖

等等。也可以切換風格。像是XMIND極簡風

XMIND經典風

預設

等。只要繪製一次圖型，切換不同圖型架構與風格即可產生不一樣的圖型，使用上十分的方便。

除此之外XMind還推出XMind Share網站功能，可讓我們把繪製好的心智圖上傳分享。也可以瀏覽他人分享的心智圖並下載。

上傳上去的心智圖也可以嵌入到自己的網站。

主要功能

  • 繪製心智圖、魚骨圖、組織圖、二維表格、流程圖、樹狀概念圖等不同架構混搭。
  • 符合完整的心智圖法操作功能，操作設定簡易，使用介面親和、圖型風格漂亮。
  • 快速主題延伸、浮動變換，及文字顏色線條圖標圖片標籤備註等元素增添豐富。
  • 內容可夾附文件檔案、超連結、圖標、圖片、音源檔等，最適於電腦化的交流。
  • 應用於腦力激盪、會議記錄、創意發想、專案管理、心得整理等，使用效益宏大。

使用特色

  1. 多重圖面的「工作簿」檔案，整合更多的資料。
  2. 下鑽上鑽的操作功能，簡化龐大圖面層次顯示。
  3. 現場錄音功能，可用於會議場景，或學習課程。
  4. 軟體介面工作場景彈性設定，可自行變更喜好。
  5. 建立專案任務資訊，檢視工作排程甘特圖畫面。
  6. 內建多款風格樣式，讓你快速引用來美化圖面。
  7. 預附精美圖標圖庫背景圖等，可自行加入喜好。
  8. 匯出png/gif/jpg/bmp圖檔。可上傳網頁分享。
  9. 可匯轉FreeMind, MindManager 的檔案使用。
  10. 可匯出.doc .ppt .txt .pdf .html 等格式檔案運用。
  11. 跨越不同系統平台使用、Java技術，流通性佳。
  12. 其它 ……

系統需求

Hardware:
  • A 800Mhz or greater processor
  • 256M RAM or greater, recommend 512M or greater.
  • 100MB free disk space for downloaded file and installaltion.
  • Internet connection for certain features (like Share) and XMind Pro.
Windows Users:
  • Windows XP/2003/Vista 32-bit.
  • Microsoft Word/Powerpoint for certain features (like Export).
Mac Users:
  • Mac OS X 10.4 and later.
  • Mac OS X 10.4 needs “Java for Mac OSX 10.4 Release 5” installed, and follow the steps in Important Notice for Mac OS X 10.4 users to modify XMind.ini file.
Linux Users:
  • GTK
  • Some features need lame(http://lame.sourceforge.net/).
  • Only Sun JRE/OpenJDK/IBM JRE/BEA JRockit are supported. Don’t use unsupported JVM.
