[C++]C++ Create GUID

Posted on 2011-10-13   |  


在C++中若有建立GUID的需求，可先建立GUID的物件實體，透過CoCreateGuid API填寫GUID到GUID的物件實體，再透過UuidToString API將GUID轉為字元陣列，像是下面這樣：



wstring GetGUID()
{
    _TUCHAR *guidStr = NULL;

GUID *pguid = new GUID;

CoCreateGuid(pguid); 

// Convert the GUID to a string
UuidToString(pguid, (RPC_WSTR*)&amp;guidStr);
delete pguid;
return wstring(guidStr);

}



完整的範例如下：



// ConsoleApplication5.cpp : Defines the entry point for the console application.
//

#include “stdafx.h”

#include <objbase.h>

#include <string>

using namespace std;

wstring GetGUID()
{
_TUCHAR *guidStr = NULL;

GUID *pguid = new GUID;

CoCreateGuid(pguid); 

// Convert the GUID to a string
UuidToString(pguid, (RPC_WSTR*)&amp;guidStr);
delete pguid;
return wstring(guidStr);

}

int _tmain(int argc, _TCHAR* argv[])
{
wstring guid = GetGUID();
wprintf(guid.c_str());
return 0;
}
