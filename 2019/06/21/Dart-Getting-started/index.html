Dart - Getting started

Posted on 2019-06-21   |  

Dart 安裝好後，不免俗的要先寫個簡單的 Hello World 程式，用以了解程式的撰寫與運行會要怎樣處理。


開啟編輯器開始撰寫程式。撰寫 main() 方法用來定義程式的進入點，main() 裡面透過 print() 將訊息顯示出來。


檔名記得以 dart 為副檔名。


程式撰寫好透過 dart 命令帶入檔名運行即可。 

dart $DartFile
