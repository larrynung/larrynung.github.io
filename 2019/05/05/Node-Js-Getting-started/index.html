Node.Js - Getting started

Posted on 2019-05-06   |  

Node.Js 安裝完後，開個 Js 檔來撰寫簡單的 Hello World 程式。


然後調用 node 命令並帶入程式位置。 

node ${file}


即可運行程式，顯示程式運行後的結果。


再來看一下稍微複雜點的 Hello World。


會在特定 Port 號起一個 http 服務，該服務會回應 “Hello World!” 字樣。


將服務啟動。


用瀏覽器訪問服務的位置，即可看到服務回應我們預期的結果。
