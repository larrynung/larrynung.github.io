Code Converter Extension v1.1

Posted on 2013-11-06   |  

在撰寫程式時，我們常常會需要做程式碼的轉換，這時我們會需要一些線上網站的輔助，像是developerfusion。Code Converter Extension試圖將developerfusion與Visual Studio結合，讓程式開發人員能在Visual Studio內就做好轉換的動作。

Code Converter Extension可直接透過Visual Studio內建的Extens and Updates安裝。

安裝完後編輯區的右鍵選單與上方EDIT工具選單內的Paste Special會新增對應的選單選項。

目前主要的功能是可以讓我們將剪貼簿內的程式碼透過developerfusion轉換後貼上，或是複製選取的程式碼後透過developerfusion轉換剪貼至剪貼簿中分享。

運行時處理的動作會顯示在輸出視窗中，對應的轉換位置也會顯示在這邊，若有需要這邊也可以直接複製後貼給他人。
