Oracle SQL Developer 預設 Query 出來的結果會在同一個 Query Result 視窗，若要讓每次 Query 都在個別的視窗，我們可以開啟 Perferences 對話框，在 [Database | Worksheet] 下勾選 `Show query results in new tabs` 選項。
