要在 Termux 安裝 Python，可透過 pkg 安裝 python 套件。  

<!-- More -->

    pkg install python

{% asset_img 1.png %}

</br>


安裝完可查閱 Python 版本做過確認，Python 命令應可正常運行，回應安裝的 Python 版本。  

    python --version
