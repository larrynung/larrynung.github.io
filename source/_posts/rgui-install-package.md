要安裝 R 語言的套件，可透過點選 RGui 的 [Packages|Install package(s)...] 主選單選項。  

<!-- More -->

{% asset_img 1.png %}

<br/>


選取境象位置。  

{% asset_img 2.png %}

<br/>


選取所要安裝的套件。  

{% asset_img 3.png %}

<br/>


指定的套件就會開始進行下載並安裝。  

{% asset_img 4.png %}

<br/>


除了透過主選單選項外，也可以在 Console 使用 'install.packages' 帶入要安裝的套件名稱，輸入完一樣選取鏡象位置。  

{% asset_img 5.png %}

<br/>


一樣可以進行指定套件的下載並安裝。  

{% asset_img 6.png %}
