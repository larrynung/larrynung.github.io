要透過 FastoRedis 寫入資料到 Redis，可在 Redis 連線後在左側的樹狀節點中找到要寫入資料的 Database，在上面按下滑鼠右鍵，選取滑鼠右鍵快顯選單中的 Create key 選單選項。  

<!-- More -->

{% asset_img 1.png %}

<br/>


在彈出的視窗中選取要寫入的資料型態、Key 的名稱、以及要寫入的值，設完後按下 OK 按鈕。  

{% asset_img 2.png %}

<br/>


指定的資料就會寫入到 Database 中。  

{% asset_img 3.png %}

<br/>


需要的話可透過點選左側的樹狀節點查看資料的內容。 

{% asset_img 4.png %}
