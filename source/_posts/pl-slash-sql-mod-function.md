MOD function 可傳回兩數相除後的餘數。  

<!-- More -->

<br/>


使用語法如下：  

    MOD( m, n )


其中 m 為被除數，n 為除數。  

<br/>


所以如果要取 5/2 的餘數，可以像這樣寫。  

    Mod(5, 2)
