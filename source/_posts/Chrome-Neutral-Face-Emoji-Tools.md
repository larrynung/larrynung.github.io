Netural Face Emoji Tools 是 Chrome 的外掛套件，可讓 Slack 支援表情符號拖曳上傳。

<!-- More -->

{% asset_img 1.png %}

<br/>


外掛套件安裝後，Slack 上傳表情符號的頁面上方會多出一塊上傳區，將要上傳的表情符號拖曳至上傳區。  

{% asset_img 2.png %}

<br/>


即完成表情符號的上傳。
