要調整 Telegram 的配色，可點選 Telegram 左上腳的按鈕展開側邊選單。  

<!-- More -->

{% asset_img 1.jpg %}

<br/>


點選 Settings 進入設定畫面。  

{% asset_img 2.jpg %}

<br/>


點選 Chat Settings 進入聊天設定畫面。  

{% asset_img 3.jpg %}

<br/>


在 Color theme 那邊選取要使用的 Color theme。  

{% asset_img 4.jpg %}

<br/>


即可調整 Telegram 的配色。  

{% asset_img 5.jpg %}
