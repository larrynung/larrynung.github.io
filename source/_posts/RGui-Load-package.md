要在 RGui 載入安裝的 R 語言套件，可點選 [Packages|Load package…] 主選單選項。  

<!-- More -->

{% asset_img 1.png %}

<br/>


選取所要載入的套件。  

{% asset_img 2.png %}

<br/>


即會載入指定的套件。  

{% asset_img 3.png %}

<br/>


除了透過主選單選項外，也可以在 Console 使用 ‘library()’ 帶入要載入的套件名稱，輸入完一樣會載入指定的套件。
